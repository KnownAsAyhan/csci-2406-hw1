# CSCI-2406 Homework 1

## Description
This program calculates the sum of integers from m to n (inclusive)
using ARM assembly without using branch instructions.

Inputs:
- r0 = m
- r1 = n

Output:
- r2 = sum

The program manually modifies the Program Counter (PC)
using calculated offsets and terminates using `svc 2`.
